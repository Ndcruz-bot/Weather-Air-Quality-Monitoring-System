# Weather & Air Quality Monitoring System

## Description
This project is a Python-based Weather & Air Quality Monitoring System developed for the Climate Data Monitoring Station in Ho Chi Minh City.

## Features
- Enter monthly environmental data
- Validate user input
- Calculate minimum, maximum, and average values
- Compare PM2.5, PM10, and NO₂ against WHO and Vietnamese standards
- Generate charts using Matplotlib
- Save reports
- Handle invalid input using exception handling

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab
